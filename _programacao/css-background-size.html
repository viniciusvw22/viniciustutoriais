Propriedade background-size

A propriedade background-size define as dimensões (largura e altura) da imagem que foi colocada usando CSS.

Sintaxe:

background: width height;

Exemplo:

          
            div#pai {
              width: 500px;
              height: 500px;
              border: 2px solid blue;
              background: url("galeria-02.jpg") no-repeat;
              background-size: 250px 250px;
            }

A diferença de width e height de background-size e width e height colocados sozinhos (no caso, 500px) é que o primeiro se refere ao width e height do elemento, enquanto que no background-size, eles se referem ao width e height da imagem colocada dentro do elemento.
