Explicando CSS: transition

transition: Quando uma propriedade muda de valor, transition faz essa mudança acontecer de forma suave.

Como usar (generalização):

    
    elemento de destino {
      (propriedade de destino): valor;
      transition: (propriedade de destino) (tempo de transição);
    }
  
    elemento de destino:hover {
      (propriedade de destino): novovalor;
    }

Como usar (exemplo):

    
    li {
      background-color: blue;
      transition: background-color 1s;
    }
    
    li:hover {
      background-color: green;
    }
  
  

    <ul type="none">
      <li>elemento de destino</li>
    </ul>
  • elemento de destino

No exemplo, toda vez que o usuário passar o mouse em cima do item da lista, a cor irá percorrer o intervalo de cores, da cor azul até chegar à cor verde. E essa transição levará 1s.
