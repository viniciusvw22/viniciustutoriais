Código-fonte, em programação, é um código escrito por um programador, e que diz ao computador o que ele deve fazer. Geralmente é composto por palavras, e por isso, o computador ainda não consegue entender este código, mas uma pessoa comum consegue.

Já código executável também é um código que diz ao computador o que ele fazer, mas é composto apenas de zeros e ums, ou seja, é um código binário. O computador consegue entender e executar este código, mas uma pessoa comum não consegue entender.

Exemplo de código-fonte, na linguagem Visual Basic.Net:

    
    Module Programa
        Sub Main()
            Console.WriteLine("Alô Mundo!")
        End Sub
    End Module

O compilador é o responsável por transformar o código-fonte em código executável
