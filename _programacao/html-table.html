Explicando os elementos de table:

caption:
define o título da tabela.
thead:
define a área de cabeçalho das colunas.
th:
define uma célula de cabeçalho, seja de linha ou coluna.
td:
define uma célula de dados.
The colspan attribute defines the number of columns a cell should span. O atributo colspan define o número de colunas que a célula deve abranger. Funciona bem quando a célula não é a última da tabela.
