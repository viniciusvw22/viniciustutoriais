Aprenda a usar o valor !important e porque evitá-lo ao máximo.

A declaração !important serve para que o valor definido para um atributo CSS não seja sobrescrito por outro valor posteriormente.

Veja um exemplo:

Código HTML: 
    
      <body>
        <div class="pai">
          <p>Esse é o texto de contido no div pai.</p>
          <div class="filho">
            <p>Esse é o texto contido no div filho, que por sua vez está contido no div pai.</p>
          </div>
            
        </div>
      </body>
Código CSS: 
	  
      div.pai {
          color: blue;
      }

      div.pai {
        color: green;
      }
Resultado:

Primeira div com classe .pai

Segunda div com classe .pai.

Como o valor 'color: green' foi definido por último, esta prevalece. Se usarmos a declaração !important em 'color: blue', teremos que a cor permanecerá azul: 
	  
      div.pai {
        color: blue !important;
      }

      div.pai {
        color: green;
      }
Resultado:

Primeira div com classe .pai

Segunda div com classe .pai.

Por que não usar a declaração !important

CSS significa Cascading Style Sheets. Vamos ler o significado de Cascading no Dicionário Cambridge:

"to pass on information by giving it to just a few people, who then give it to more people; to be passed on in this way:"

>> Distribuir informações para apenas algumas pessoas, que então passarão para mais algumas outras pessoas da mesma forma, e assim sucessivamente.

É como em uma empresa, que a informação é passada de um chefe para seu subordado mais próximo, que vai passagem para alguns poucos subordinados próximos dele, assim por diante, até a informação chegar até o nível mais baixo. Portanto, CSS funciona por hierarquia. Devemos seguir esta regra, portanto, evitando usar !important, para não ter dor de cabeça mais para frente.
