Ponteiros em C

Para entender ponteiros, primeiro é preciso saber:

  1. O que é uma variável
  2. O que é um endereço de memória

• Variáveis são espaços vazios na memória que podem armazenar valores.

• Endereço de memória é simplesmente o local onde uma variável está, assim como o endereço de uma casa em uma rua.

• Agora sim, ponteiro é uma variável que armazena o endereço de outra variável.

Assim no exemplo, temos que a variável a é um ponteiro que armazena o endereço da variável d, logo dizemos que a aponta para d. 

          
            int d = 30;

            int *a = &d; /// o ponteiro a recebe o endereço da variável d.
            
            printf("O peso do meu cachorro é de %d kg", *a);
            
            /// Saída: 
            O peso do meu cachorro é de 30 kg
          
        
 
           int d = 30;

Aqui eu declarei uma variável d, do tipo inteiro (int), e atribuí a ela o valor 30 (kg).

           
            int *a = &d; 
           
          /// o ponteiro a recebe o endereço da variável d, OU, o ponteiro a 
          aponta para a variável d.

Aqui eu declarei o ponteiro a, e em seguida atribuí o endereço da variável d a ele (== 1006).

• operador * : serve para declarar um ponteiro, isto é, diz que esta variável armazena endereços de memória.

• operador & : retorna o endereço de memória de uma variável. No caso, o endereço vale 1006.

• obs.: int significa que o ponteiro a aponta para uma variável do tipo inteiro. Se o ponteiro apontasse para uma variável do tipo double, deveríamos usar double na declaração.

 
            printf("O peso do meu cachorro é de %d kg", *a);

• operador * : também é usado para retornar o valor de uma variável que está em um certo endereço. Como a vale 1006, *a retornará o valor que está na variável de endereço 1006, valor que no caso é 30 (kg).
Saída: O peso do meu cachorro é de 30 kg
